TerraVolt — Mycelium Tile Demo Website

This repository hosts the official TerraVolt Demo Website, showcasing a fully animated explanation of how mycelium‑based tiles regenerate soil and capture solar energy.  
The site is lightweight, fast, and built entirely with HTML, CSS, and JavaScript, making it ideal for presentations, science fairs, investor demos, and educational use.

The website is deployed using GitHub Pages, allowing anyone to open it instantly on any laptop with a free domain.

About the Project

TerraVolt explores a nature‑inspired energy system where mycelium networks act as both:

- A soil regeneration engine, improving soil health over time  
- A solar‑energy capture and distribution network, visualized through glowing particle flows  

This demo website includes a custom‑built animation that illustrates:

- Sunlight pulsing into the soil  
- Soil layers healing and brightening  
- Mycelium networks expanding dynamically  
- Energy particles flowing through the fungal network
